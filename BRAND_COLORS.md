# DIGM ◉rigins Brand Colors

## Color Palette

### Primary Colors

**Grayscale Monochrome Scheme**:
- **Ivory/Off-White**: `#F5F5F0`, `#FAFAF5`
- **Silver Gray**: `#C0C0C0`, `#D3D3D3`
- **Light Gray**: `#A0A0A0`, `#B5B5B5`
- **Charcoal Gray**: `#2F2F2F`, `#404040`
- **Dark Slate**: `#1A1A1A`, `#252525`

### Texture Overlay

**Marble/Stone**:
- Subtle veining in light gray tones
- Stone-like surface texture
- Matte finish (not glossy)
- Premium, sophisticated feel

---

## Usage Guidelines

### Do
- ✓ Use monochrome palette exclusively
- ✓ Apply subtle marble textures
- ✓ Maintain high contrast for readability
- ✓ Use ivory for backgrounds
- ✓ Use charcoal for text/icons
- ✓ Add subtle silver accents

### Don't
- ✗ Add bright colors
- ✗ Use gradients (except marble texture)
- ✗ Over-saturate textures
- ✗ Make designs too busy
- ✗ Use glossy/shiny effects

## Accessibility

### Contrast Ratios
- Ivory on Charcoal: 15.5:1 (AAA)
- Charcoal on Ivory: 15.5:1 (AAA)
- Gray on Charcoal: 3.8:1 (AA)
- Silver on Ivory: 1.8:1 (No pass - use sparingly)

### Recommendations
- Use Charcoal (#2F2F2F) for primary text
- Maintain minimum 4.5:1 contrast for readability
- Test with grayscale filter to ensure differentiation
- Provide high contrast mode option

## Material & Texture

### Marble/Stone Application
- Apply as subtle overlay (10-20% opacity)
- Use on backgrounds and key elements
- Maintain monochrome palette
- Avoid overwhelming the design

### Logo Treatment
- Primary logo: Charcoal with marble texture
- Variant: Silver with marble texture
- Monochrome variant: Black with marble texture
- Reverse variant: Ivory with charcoal marble texture
